Prime Hoopers

Prime Hoopers is a static web application created and designed by Noli II B. Albay that is developed with the help of different web technologies such as HTML and CSS. The purpose of this project is to let us fully familiarize with different CSS syntax and to develop our creativity in designing web application. On this web application, you can view different articles that are about on the current event on NBA. It is also a recruitment web application that encourages people to join on different pieces of training we offer that will help them improve their skills on basketball court assisting them with our different expert trainers.
